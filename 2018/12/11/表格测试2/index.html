算数运算符

运算符 说明 使用举例
+ 加法 expr $a + $b
- 减法 expr $a - $b
* 乘法 expr $a \* $b
/ 除法 expr $a / $b
% 取余 expr $a % $b
= 赋值 a = $b 将变量b的值赋给a
== 判相等，相同为true [ \$a == \$b ]
!= 判不等，不等为true [ \$a != \$b ]

关系运算符

关系运算符只支持数字，不支持字符串，除非字符串的值是数字
假设a=20; b=30
运算符 | 说明 | 使用举例
— | — | —
-eq | 两个变量相等 返回 true | [ \$a -eq \$b ] 返回false
-ne | 两个变量不相等 返回 true | [ \$a -ne \$b ] 返回true
-gt | 左变量 > 右变量 返回true | [ \$a -gt \$b ] 返回false
-lt | 左变量 < 右变量 返回true | [ \$a -lt \$b ] 返回true
-ge | 左变量 >= 右变量 返回true | [ \$a -ge \$b ] 返回false
-le | 左变量 <= 右变量 返回true | [ \$a -le \$b ] 返回true
注意
表达式要放在方括号之间，并且要有空格，[ \$a -gt \$b ]。

逻辑运算符

假设a=20;b=30

运算符 说明 使用举例
&& 逻辑上的 与 [[ \$a -lt 100 && \$b -gt 100 ]] 返回false
\ \ 逻辑上的 或 [[ \$a -lt 100 \ \ \$b -gt 10]]返回false

注意
表达式要放在双(double)方括号之间，并且要有空格[[ \$a -lt 100 && \$b -gt 100 ]].
